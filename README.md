# shortest-path-algorithm
A fast algorithm that can simultaneously calculate the number and length of the shortest paths between all pairs of nodes in a network.

If you find this algorithm helpful, please cite "Tan D, Deng Y, Xiao Y, et al. Shortest path counting in complex networks based on powers of the adjacency matrix[J]. Chaos: An Interdisciplinary Journal of Nonlinear Science, 2024, 34(10): 101104 ."
